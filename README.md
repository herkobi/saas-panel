## Herkobi Panel SaaS

Herkobi Panel SaaS; SaaS uygulama geliştirmek isteyen kişi/kurumlar için müşteri ve paket yönetimi için hazır altyapı sunar.

Müşterilerinizin ödeme durumlarını, kullandıkları paketleri, paketlerin özelliklerini vb. işlemleri yönetmenizi sağlayarak motivasyonunuzu sadece yazılımınızı geliştirmenize yönelterek sizlere yardımcı olur.

Ücretsiz ve açık kod olarak dağıtılan Herkobi Panel SaaS ile özellikle micro-saas uygulamaları geliştirmeniz hız kazanmaktadır. Sistemimiz sub-domain ekleme, kullanıcı bazlı özel paket oluşturma, kullanıcı bazlı zaman dilimi ve dil seçimi gibi özelliklerle genişletilecektir.
